# Student Management System

## Description
This is a mini project made in C++.
It helps manage student records like:
- Add Student
- Display Students
- Search Student
- Calculate Grade

## Features
- Store student data
- Calculate grades using marks
- Easy menu system
- File handling support

## Technologies Used
- C++
- VS Code
- GitHub

## How to Run
1. Open project in VS Code
2. Compile the code
3. Run the program

Example:

```bash
g++ main.cpp -o main
./main
```
